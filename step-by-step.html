Langkah-langkah Menggunakan Aplikasi FAST

Ikuti setiap langkah berikut untuk melakukan absensi wajah dengan benar.

1. 📦 Persiapan Awal

✅ Sistem yang Dibutuhkan:
OS: Windows 10 atau 11 (64-bit)
Python 3.9+ (harus diinstal)
Webcam aktif
Akses internet (untuk notifikasi ntfy.sh jika digunakan)

2. 🔽 Download dan Install Python

1. Kunjungi https://python.org/downloads
2. Pilih versi terbaru (misalnya Python 3.12.x)
3. Saat instalasi:
- Centang “Add Python to PATH”
- Klik “Install Now”

3. 📁 Unduh Proyek FAST

1. Buka GitHub: github.com/abyfromheaven/FAST
2. Klik tombol Code → Download ZIP
3. Ekstrak ZIP ke folder mana saja (misalnya C:\FAST)

4. 🧪 Install Library yang Dibutuhkan

Buka Command Prompt (CMD), lalu jalankan:
cd C:\FAST
pip install -r requirements.txt

Jika tidak ada file requirements.txt, gunakan:
pip install customtkinter opencv-python pyttsx3 requests

5. 🎤 Test Text-to-Speech (Opsional)

FAST memakai pyttsx3 untuk suara notifikasi.
Windows sudah mendukung ini secara bawaan lewat SAPI5,
jadi tidak perlu install tambahan.

6. 🚀 Jalankan Aplikasinya

Masih di Command Prompt:
python main.py

Tunggu beberapa saat hingga GUI aplikasi terbuka.
Kamu akan melihat antarmuka modern dari FAST.

7. 🎯 Fitur yang Bisa Digunakan

Menu – Fungsi:

🎥 Rekam Wajah Siswa: Menyimpan wajah siswa untuk pelatihan model
🎥 Rekam Wajah Guru: Menyimpan wajah guru untuk pelatihan model
🧠 Latih Model: Melatih model pengenalan wajah
📝 Absensi Siswa: Deteksi wajah siswa dan catat waktu kehadiran
📝 Absensi Guru: Sama seperti siswa, untuk guru
🔔 Notifikasi: Kirim notifikasi otomatis ke ntfy.sh (jika diaktifkan)

8. 🔔 Notifikasi Real-Time (Opsional)

Jika menggunakan ntfy.sh, notifikasi akan dikirim ke channel khusus.

1. Buka file main.py
2. Temukan dan ubah bagian berikut:
ntfy_url = "https://ntfy.sh/fast-absensi-channel"
3. Buka ntfy.sh/fast-absensi-channel di HP/browser untuk menerima pesan.

9. ❗ Jika Terjadi Masalah

Masalah – Solusi:

📷 Kamera tidak muncul – Tutup aplikasi lain yang pakai webcam
🔇 Tidak ada suara – Periksa volume & pastikan SAPI aktif
❌ "Module not found" – Jalankan pip install untuk modul tersebut
📂 File tidak ditemukan – Jalankan CMD di folder yang benar (cd C:\FAST)

10. 💡 Tips Tambahan

🔐 Jalankan sebagai Administrator jika ada error permission
⚠️ Jangan ubah nama file haarcascade_frontalface_default.xml
💡 Untuk hasil terbaik, pastikan wajah terang dan terlihat jelas di kamera
